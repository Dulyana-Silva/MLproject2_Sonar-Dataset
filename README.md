# SONAR-Based Rock vs Mine prediction.

When a submarine navigating through a shallow, to "see" underwater where light doesn't reach, it sends out a sonar pulse (sound wave). This sound hits an object on the seafloor and bounces back.

<p align = "center"><img width="300" height="300" alt="image" src="https://github.com/user-attachments/assets/1c2225cd-8f55-4262-8469-78e46a3abbae"/></p>

## Problem Statement.
-  In this project, we're predicting whether a submerged object detected by a submarine's sonar is a natural rock or a naval mine (a hidden underwater explosive).
-  Underwater metal cylinders are often designed to look like rocks or are hidden near them to avoid detection.

## The Danger.
If a submarine thinks a mine is just a rock, it might strike the mine, leading to a fatal explosion and loss of life and in the other hand if submarine thinks a rock is a mine, it will waste time and resources taking unnecessary "evasive measures".

## About the Dataset

- The dataset contains 208 samples.
- 60 numeric features representing sonar signal intensities.
- Target labels (R- Rock, M- Mine)
